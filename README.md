# tvmaze-scraper

[![Build Status](https://travis-ci.org/rodrigocmoreira/tvmaze-scraper.svg?branch=master)](https://travis-ci.org/rodrigocmoreira/tvmaze-scraper)

## To test on Windows

```sh
npm run test_windows
```

or

```sh
npm run mocha_windows
```

## To test on Unix OS

```sh
npm test
```

or

```sh
npm run mocha
```

## To run on Windows

```sh
npm run start_windows
```

## To run on Unix OS

```sh
npm start
```
